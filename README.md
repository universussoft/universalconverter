# 🔄 Universal Unit Converter

> **Free. No ads. No sign-up. No external dependencies.**
> A single self-contained HTML file with 1300+ units across 125 categories — works fully offline.

## ✨ Features

- **125 categories** — from length and mass to ideal gases, RGB colours, IP networks and spectroscopy
- **1300+ units and reference values**
- **6 languages** — 🇵🇹 PT · 🇬🇧 EN · 🇪🇸 ES · 🇫🇷 FR · 🇩🇪 DE · 🇨🇳 ZH (switch instantly, no reload)
- **Two interface modes:**
  - **Numeric converter** — for mathematically related units (e.g. km ↔ miles ↔ feet)
  - **Lookup / equivalence table** — for non-linear systems (e.g. clothing sizes, tyre codes, paper formats)
- **Fully offline** — one single `.html` file, no server, no API, no internet required after download
- **Dark mode** — native dark theme, easy on the eyes
- **Zero dependencies** — no frameworks, no npm, no build step. Just open in a browser.

## 🚀 Quick Start

```bash
# Option 1 — just open the file
open conversor-universal.html

# Option 2 — serve locally (optional)
npx serve .
# or
python3 -m http.server 8080
```

No installation. No configuration. Works in any modern browser.

## 🏗️ Architecture

```
conversor-universal.html          ← the entire application (single file)
│
├── <style>                       ← CSS — dark theme, responsive, lookup UI
│
├── <body>                        ← HTML — converter card, lookup card, interest card
│
└── <script>
    ├── TRANSLATIONS (C object)   ← category names in all 6 languages
    ├── CONSTANTS                 ← TITLE, SUBTITLE, FOOTER, DEV_TITLE, DEV_DESC...
    ├── CATEGORIES object         ← all 125 categories with units / rows
    ├── Special converters        ← temp, sound, fuel, pH, pace, altitude, RGB, CIDR...
    ├── Lookup renderer           ← renderLookup(), makeChip(), buildResultBar()
    ├── Interest calculator       ← calcInterest() — interactive compound interest
    └── UI engine                 ← populateSelects(), convert(), applyLang()...
```

### Category types

| Type | Description | Example |
|---|---|---|
| `u: {}` | Numeric — linear factor conversion | Length, Mass, Energy |
| `special: "temp"` | Custom math, non-linear | Temperature (Kelvin↔Rankine) |
| `special: "fuel"` | Inverse units (L/100km) | Fuel Economy |
| `special: "sound"` | Logarithmic (dB) | Sound Level |
| `special: "ph"` | Log₁₀ based | pH |
| `special: "pace"` | min:sec formatting | Running Pace |
| `special: "rgb"` | Multi-format parsing | Colours |

## 📐 Interface Modes

### Numeric Converter
Used for all mathematically related units. Enter a value, select source and target units — all other conversions appear as clickable cards below.

### Lookup / Equivalence Table
Used for systems that are not mathematically linear — sizing charts, codes, lookup values. Shows a single dropdown; selecting a size displays:
- The measurement in the correct unit (mm, cm, inches, etc.)
- All equivalent sizes grouped by **exact match**, **close match**, and **all options**

Special display logic per category:
- **Tyres** — shows width, sidewall height, rim size decoded from the code (e.g. 205/55R16)
- **Drill bits / Screws** — shows diameter in mm and fractional inches
- **AWG wire** — shows mm² and equivalent diameter
- **Paper** — shows dimensions in mm, cm, and inches with area in m²
- **Monitor / TV** — shows cm + inches + typical resolution for that diagonal
- **Colour temperature** — shows Kelvin with a descriptive label (warm/neutral/daylight/cool)

### Interest Calculator
Dedicated interactive card for 💰 Juros. Three editable fields (capital, rate, period) update all results in real time:
- Simple vs compound final amount
- Total interest both methods
- Monthly and daily equivalent rate
- Exact doubling time (and Rule of 72 approximation)
- Present value of €1 at maturity
- Year-by-year evolution table

## 🛠️ Development Status

> ⚠️ **This tool is actively under development.**
> New fields, categories and features are added regularly. Some values may still be incomplete or require validation.

### Known limitations
- **💰 Interest** — uses fixed reference scenario (€1000, 5%, 10yr) as base; interactive inputs override this at runtime
- **📍 GPS** — UTM conversion is zone-only (no full easting/northing)
- **⚗️ Ideal Gas** — fixed at standard scenario (1 mol, 25°C, 1 atm); full PVT solver not yet implemented
- **🔬 Moles** — defaults to H₂O (18.015 g/mol); custom molar mass input is partial
- **📊 Percentages** — uses simplified defaults; full interactive calculator coming
- **💊 Pharma Dose / Conc.** — reference values only, not a medical calculator
- **🎲 Probability** — linear factor table only; full Bayesian calculator planned

### Roadmap
